# Tips

Git Clone 的时候会根据协议不同而读取不同的代理。

如果使用 `export ALL_PROXY=xxx` 的话，只会影响使用 http 协议 clone 的方式。如果是要 clone ssh 协议的 Repo，这个声明方式不会起作用。

如果要用对 ssh 协议起作用，还是需要配置 ssh config 的 ProxyCommand。详见 `man ssh_config`
